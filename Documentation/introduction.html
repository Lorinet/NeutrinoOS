Introduction to Neutrino

What is Neutrino

Neutrino is an application development platform and operating system built for embedded devices, personal computers, IoT appliances, smart wearable devices and more.
The core of the system is based on a virtual machine (VM) which executes Neutrino bytecode.
The Neutrino bytecode is the compact encoding of the Neutrino instruction set architecture.
Bytecode can be created using the Intermediate Language Assembler (ntrasm) tool, which assembles programs written in Neutrino IL.
The Neutrino Runtime VM is available for Windows and for Linux, and allows Neutrino apps to be run concurrently with native apps.
The standalone Neutrino Core OS is composed of the Neutrino VM running on top of a minimal Linux kernel, and is designed for embedded systems and special-purpose computers.

Features

The platform supports various powerful features, such as dynamic linking, system calls and dynamic memory management.
Dynamic memory management allows the developer to allocate variables with dynamic sizes, without dealing with pointers or memory locations, because the memory allocation is managed by the VM.
This also makes the platform secure to memory leaks and ensures that any code would manage memory optimally.
The user code communicates with the system (Neutrino VM) through system calls. The system calls perform a variety of low level OS functions (e.g. for I/O management, file system accesss etc.).
